# Randomy

Fun project: *Randomy* is a simple application that selects a random element from a set of user inputs.

## App features
* Add and Item
* Clear Item
* Swipe to delete item
* Empty list of items
* Dynamic theming
* Theme persistance
* Choose random item from the list
